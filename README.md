# QuoteMachineCS
Add/Remove quote ("> ") to/from text in clipboard.
The current hotkey settings:

    Ctrl+F8: Remove Quote
    Ctrl+F9: Add Quote

ex.) Add quote

Before in clipboard:

    aaa
    bbb


After in clipboard:

    > aaa
    > bbb

This application is running in task tray. You can right-click the icon in the task tray to exit.

Platform: Windows10.
